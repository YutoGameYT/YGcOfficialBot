## 🔧コマンド🔧
```y.avator```
あなたのアバターを表示します

```y.coin```
コインゲームができます。表裏がでるよ

```y.guns```
ランダムで銃が出てくるよ。これでゲームの縛りしたら楽しそう

```y.help```
ヘルプを表示します

```y.info```
概要を表示します

```y!invites```
サーバー内のすべての招待リンクを表示します

```y.members```
YGc公式Bot EXが参加しているサーバーの合計メンバー数を表示します

```y.ping```
Botが死んだと思ったら入力しましょう。Pong!と帰ってきたら生きてます

```y.servers```
YGc公式Bot EXが参加している合計のサーバー数を表示します

```y.website```
ソースコードのリンクを表示します
